Istruzioni per l'uso


Glossario

In questa pagina la terminologia presente nel database viene mostrata in forma di semplice glossario.

Ogni riquadro riporta un set di informazioni inserite nel database dall’amministratore o dagli utenti. Gli elementi sono ordinati alfabeticamente secondo i lemmi.

La barra di ricerca consente di cercare una stringa di testo in tutti i campi del database. Utilizzando le funzioni di ricerca avanzate è inoltre possibile cercare una stringa di testo solo in alcuni campi del glossario.

Questo strumento è pensato per visionare la terminologia in maniera semplice ed è ottimizzato per essere utilizzato durante le attività di emergenza.


Aggiungi terminologia

A questa pagina è possibile suggerire l’aggiunta di nuova terminologia nel database.

Un apposito form consente di inserire una voce di terminologia alla volta.

La terminologia inserita dagli utenti sarà soggetta a revisione da parte dell'amministratore del sito prima di essere mostrata nel glossario e nel metaglossario.


Aggiungi terminologia in massa

A questa pagina è possibile suggerire l’aggiunta di nuova terminologia nel database.

Un apposito fileform consente di inserire più voci di terminologia alla volta, oppure di un intero glossario.

Per inserire la nuova terminologia con questo strumento è necessario scaricare un foglio elettronico, compilarlo come indicato e poi ricaricarlo utilizzando l’apposito fileform.

La terminologia inserita dagli utenti sarà soggetta a revisione da parte dell'amministratore del sito prima di essere mostrata nel glossario e nel metaglossario.


Esporta terminologia

Da questa pagina è possibile esportare la terminologia presente nel database, sia attraverso il download diretto di un foglio elettronico, sia attraverso un servizio di API in formato JSON.

L’esportazione della terminologia tramite API costituisce modalità di condivisione in linked data, garantendo così l’aggiornamento in tempo reale delle informazioni dal database sorgente.
